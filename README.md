# express-mysql

# 使用技術
- Express.js
- MySQL
- Docker

# 作成手順
1. Express.jsのセットアップ
   1. `mkdir express-mysql & cd express-mysql`
   2. `npm init -y`
   3. `npx express-generator --view=ejs .`
   4. `npm install`
   5. `npm start`
2. Dockerのセットアップ
   1. Dockerfileの作成
   2. docker-compose.ymlの作成
3. コーディング
4. 起動
